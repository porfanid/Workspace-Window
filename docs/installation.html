Installation

To install the Window Workspace Manager, follow these steps:

  1. Clone the Repository:

    git clone https://github.com/yourusername/window_workspace_manager.git cd window_workspace_manager

  2. Create a Build Directory:

    mkdir build cd build

  3. Run CMake to Configure the Project:

    cmake ..

  4. Compile the Project:

    make

    After building, the executable will be located in the build directory.
